# hyprland-config
Minhas configurações do Hyprland que utilizo no meu Archlinux

🌌 Hyprland Dotfiles | ThinkPad X220

Uma configuração minimalista e funcional focada em produtividade acadêmica, escrita técnica e emulação retrô. Este repositório serve como o "coração" do meu caderno digital portátil no meu ThinkPad X220.
💻 Hardware: The Cult Classic

    Modelo: ThinkPad X220

    SO: Arch Linux (Kernel LTS)

    Disco: 120GB SSD (Otimizado para baixo consumo)

    Interface: Hyprland (Wayland)

🛠️ Stack de Software

Este sistema prioriza ferramentas via terminal e sincronização constante para garantir que minhas anotações de engenharia estejam sempre seguras.

    Editor de Código: Neovim (Configuração via Lua)

    Notas & Equações: Obsidian (Fluxo de trabalho focado em Markdown e LaTeX)

    Terminal: Kitty (Configurado com cores Synthwave)

    Navegação: Firefox (Wayland native)

    Versionamento: Git + GitHub (Sincronização entre máquinas)

    Entretenimento: RetroArch (SNES, N64, NES)

🎨 Estética & Workflow

A interface segue o estilo Synthwave/Neon. No X220, o foco é maximizar o espaço de tela de 12.5" através de janelas dinâmicas e atalhos de teclado eficientes.

Atalho,Ação
Super + Q,Terminal (Kitty)
Super + E,Editor (Neovim)
Super + O,Notas (Obsidian)
Super + G,Git Status (Script customizado)
