# hyprland-config
Minhas configurações do Hyprland que utilizo no meu Archlinux

# 🌌 Hyprland Dotfiles | ThinkPad X220

- Uma configuração minimalista e funcional focada em produtividade acadêmica, escrita técnica e emulação retrô. Este repositório serve como um acesso rápido das configurações que utilizo no meu laptop ThinkPad x220.

💻 Hardware:
    Modelo: ThinkPad X220
    SO: Arch Linux (Kernel LTS)
    Disco: 120GB SSD (Otimizado para baixo consumo)
    Interface: Hyprland (Wayland)

🛠️ Stack de Software
Este sistema prioriza ferramentas via terminal por conta do baixo poder computacional.

```
    Editor de Código: Neovim (Configuração via Lua)
    Anotações: Obsidian (Fluxo de trabalho focado em Markdown e LaTeX)
    Terminal: Kitty
    Navegação: Firefox (Wayland native)
```

# 🎨 Estética & Workflow
- A interface segue o estilo Synthwave/Neon.
    - Alguns exemplos dos atalhos que eu utilizo:
```
Super + Return, Terminal (Kitty)
Super + A, Menu
Super + F, Fullscreen
Super + Space, File Manager (Dolphin)
```

# Fonte
- Para configurar a fonte, mova o arquivo dentro de fonts para esse diretório
/usr/share/fonts/TTF

- Execute o comando abaixo:
```bash
sudo fc-cache -fv
```




# Observação Técnica:
- Além de utilizar o repositório, certifique-se de instalar todos os pacotes necessários para funcionar.
